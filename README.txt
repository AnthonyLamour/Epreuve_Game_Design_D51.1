Ce repository contient l'épreuve de game design de l'élève Anthony LAMOUR, ainsi que la présentation powerpoint l'accompagnant et la documentation du projet (cahier des charges, Game concept, GDD).

Le code source du prototype, la documentation et la présentation sont sur des branches à part de ce git.

L'executable du projet a été envoyer dans le mail avec le lien de ce git.

Le projet est multijoueur et requière 2 à 4 manette (initialement prévu pour 4 joueurs)

Le projet tourne avec Unity 2020.1.14f1
